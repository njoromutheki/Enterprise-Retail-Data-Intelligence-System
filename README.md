# Enterprise-Retail-Data-Intelligence-System
📌 Project Overview

The Enterprise Retail Data & Intelligence System (ERDIS) is a full-scale relational database project designed to simulate a multi-branch retail enterprise environment.

The system centralizes operational data including:

Branch management

Employee records

Customer transactions

Product catalog & categories

Supplier relationships

Inventory tracking

Sales & payments

The objective is to transform raw transactional data into structured business intelligence.

🎯 Business Problem

Retail enterprises often struggle with:

Fragmented sales records

Manual and delayed reporting

Poor visibility into branch performance

Inaccurate inventory tracking

Limited insight into customer purchasing behavior

This project addresses these issues by designing a normalized, scalable SQL database that enables real-time analytics and performance monitoring.

🧠 Technical Objectives

This project demonstrates:

Advanced relational database design

Data normalization (1NF, 2NF, 3NF)

Primary & foreign key relationships

Transactional modeling (Sales & Sales Details)

Aggregation & analytical queries

Views for reporting abstraction

Stored procedures for automation

Triggers for business logic enforcement

Index optimization for performance

🛠 Technologies Used

MySQL

MySQL Workbench

SQL (Advanced Querying & Optimization)

Git & GitHub (Version Control)

📊 Key Analytical Capabilities

The system supports:

Total revenue analysis

Branch performance comparison

Monthly & yearly sales trends

Best-selling products identification

Customer lifetime value calculation

Inventory turnover analysis

Profit margin computation
